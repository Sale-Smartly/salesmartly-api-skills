# 参考资料

这些文档供 **AI 在执行任务时按需加载**，包含 API 规范、认证规则、字段说明等。

---

## 📚 参考资料列表

| 文档 | 用途 | 大小 |
|------|------|------|
| [authentication.md](./authentication.md) | SaleSmartly API 认证与签名规则 | 1.8KB |

---

## 🎯 何时加载

AI 在以下场景会加载这些文档：

- **首次调用 API** → 加载 `authentication.md` 了解签名规则
- **遇到认证错误** → 重新参考 `authentication.md` 检查签名步骤
- **需要扩展 API** → 参考认证规则编写新脚本

---

## 📁 与 docs/ 的区别

| 目录 | 用途 | 读者 |
|------|------|------|
| `references/` | API 规范、认证规则、字段说明 | **AI 执行任务时** |
| `docs/` | 开发者指南、计划、建议 | **人类开发者** |

---

## ➕ 添加新参考资料

当需要添加新的参考资料时：

1. **API 端点对照表** → `api-endpoints.md`
2. **错误码说明** → `error-codes.md`
3. **字段映射详情** → `field-mapping.md`

**原则**：只放 AI 执行任务时真正需要参考的内容，保持精简。
