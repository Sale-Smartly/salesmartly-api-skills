# 场景：查询客户

## 用户可能会这样说
- "查一下有哪些客户"
- "列出所有 VIP 客户"
- "最近 7 天新增的客户"
- "帮我找一下张三的客户信息"
- "看看活跃客户有哪些"

## AI 思考过程

1. **识别意图**：用户想查询客户列表
2. **提取筛选条件**：
   - 有时间范围？→ `--days`
   - 有标签？→ `--tags`
   - 有关键词？→ `--keyword`
3. **选择脚本**：`query-customers.py`
4. **设置分页**：默认 `--page-size 20`（避免返回太多）
5. **格式化输出**：表格展示关键信息

## 调用命令

### 查最近 7 天新增客户
```bash
python3 scripts/query-customers.py --days 7 --page-size 20
```

### 查 VIP 客户
```bash
python3 scripts/query-customers.py --tags "VIP" --page-size 20
```

### 查特定关键词
```bash
python3 scripts/query-customers.py --keyword "张三" --page-size 20
```

### 组合筛选
```bash
python3 scripts/query-customers.py --days 30 --tags "VIP,意向客户" --page-size 50
```

## 错误处理

| 错误情况 | AI 应该回复 |
|---------|-----------|
| 没有符合条件的客户 | "没有找到符合条件的客户，您可以尝试放宽筛选条件" |
| 标签不存在 | "未找到标签 'VIP'，可用标签：意向客户、已成交、VIP" |
| 返回数据太多 | "找到 500 个客户，已显示前 20 个。需要我分页显示或添加筛选条件吗？" |
