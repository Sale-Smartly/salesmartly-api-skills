# 场景：查询销售数据

## 用户可能会这样说
- "查一下昨天的销售数据"
- "昨天卖得怎么样"
- "昨天的成交情况"
- "看看昨天的转化"

## AI 思考过程

1. **识别意图**：用户想查看销售统计数据
2. **确定时间**：用户说"昨天"→ 使用 `--yesterday` 参数
3. **选择脚本**：销售数据 → `daily-sales-report.py`
4. **执行命令**：调用脚本获取数据
5. **格式化输出**：用表格展示关键指标

## 调用命令

```bash
cd skills/salesmartly-api-skills
python3 scripts/daily-sales-report.py --yesterday
```

## 预期输出格式

```
📊 每日销售报告 (2026-03-16)
========================================
总咨询数：156
成交客户数：23
转化率：14.7%
新增客户：45
```

## 错误处理

| 错误情况 | AI 应该回复 |
|---------|-----------|
| API 返回空数据 | "昨天还没有销售数据，可能是周末或节假日" |
| 配置错误 | "请先配置 api-key.json 文件，填入您的 API Key 和 Project ID" |
| 网络错误 | "连接 SaleSmartly 失败，请检查网络连接" |
| 参数错误 | "抱歉，我没理解时间范围，您是想查昨天、今天还是最近 7 天？" |

## 扩展场景

### 查今天的数据
```bash
python3 scripts/daily-sales-report.py --today
```

### 查最近 7 天
```bash
python3 scripts/customer-stats.py --days 7
```

### 查指定日期
```bash
python3 scripts/daily-sales-report.py --date 2026-03-15
```
